API Rest diseñada para la gestión de usuarios, contenidos y otras entidades relevantes para una plataforma de streaming online. Permite operaciones básicas como alta, baja y modificación de entidades, así como visualización de estadísticas relevantes al negocio. Diseñada en base al patrón Entidad > Controlador > Servicio > Repositorio, utilizando lenguaje Java y framework Spring Boot, con persistencia de datos en MySQL.
